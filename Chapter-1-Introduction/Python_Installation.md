# Python Installation Guide

Python is a free and open-source programming language. Before writing Python programs, you need to install Python on your computer.

## Step 1: Download Python

1. Open your web browser.
2. Visit the official Python website:

   * https://www.python.org
3. Click on **Downloads**.
4. Download the latest stable version of Python for your operating system.

## Step 2: Install Python on Windows

1. Run the downloaded installer.
2. Check the box:

```text
Add Python to PATH
```

3. Click **Install Now**.
4. Wait for the installation to complete.
5. Click **Close**.

## Step 3: Verify Installation

Open Command Prompt and run:

```bash
python --version
```

Output example:

```text
Python 3.13.5
```

You can also check pip:

```bash
pip --version
```

Output example:

```text
pip 25.1.1
```

## Step 4: Open Python Interpreter

In Command Prompt, type:

```bash
python
```

Example:

```python
>>> print("Hello, World!")
Hello, World!
```

To exit:

```python
exit()
```

## Step 5: Install VS Code

1. Download Visual Studio Code from:

   * https://code.visualstudio.com
2. Install VS Code.
3. Open VS Code.
4. Install the **Python Extension** by Microsoft.

## Step 6: Run Your First Python Program

Create a file named:

```text
hello.py
```

Write:

```python
print("Hello, Python!")
```

Run the file:

```bash
python hello.py
```

Output:

```text
Hello, Python!
```
